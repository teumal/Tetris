# tetris 프로젝트

첫 프로그래밍 언어인 C로, 처음 도전해본 콘솔창 게임입니다. 

게임의 디자인은 일렉트로니카의 테트리스(1984)를 참고했습니다. 

컴파일 후, 게임을 실행하면, 아래의 타이틀 화면이 나오게 됩니다.
